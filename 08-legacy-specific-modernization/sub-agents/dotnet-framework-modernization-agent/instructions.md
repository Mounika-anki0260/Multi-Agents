# .NET Framework Modernization Agent Instructions

## Analyze

- Solution files, projects, target frameworks, NuGet packages, WebForms pages, MVC controllers, WCF/ASMX services, Windows Services, config transforms, IIS dependencies, authentication, and database access.
- Compatibility with modern .NET, package blockers, framework-only APIs, session state, app domains, COM interop, and hosting dependencies.
- Candidate approaches: in-place upgrade, side-by-side migration, API-first extraction, UI rebuild, containerization, or service replacement.

# Legacy Modernization Questionnaire Agent Instructions

## Questions To Ask

1. Which legacy technology family is in scope?
2. Is the goal to rehost, replatform, refactor, rearchitect, rebuild, replace, retire, or wrap with APIs?
3. Which artifacts are available for analysis?
4. Which screens, jobs, reports, files, APIs, database objects, or interfaces are business critical?
5. Which compatibility constraints must be preserved?
6. What target platform, stack, database, security, and deployment constraints exist?
7. What timeline, release approach, and outage tolerance are expected?
